## Section Allocations

---

### WHAT — State the Recommended Allocation (1% rubric weight · 10.5% of brief marks)

- **Word count:** 40–55 words (one bullet)
- **Tables:** 0 — state weights inline as "STI X% / MTG Y% / LTG Z%"
- **Figures:** 0
- **Depth:** SURFACE — one declarative statement; zero justification

#### Must Include
1. **Exact allocation percentages** — STI / MTG / LTG weights, summing to 100% (e.g., "15% STI / 35% MTG / 50% LTG")
2. **Macro regime framing in one clause** — name the scenario driving the recommendation (e.g., "Under a disinflationary base case with CPI at 2.7% and gradual RBA easing…")
3. **Dominant tilt named** — which pool gets the largest share and the one-phrase reason (e.g., "tilting toward long-term growth for higher real returns")
4. **Headline portfolio metrics** — forecast gross return and estimated risk inline (e.g., "delivering a forecast 8.7% p.a. at ~8.1% risk")

#### Citations Required
- **0** — no citations in this section; it is a pure declaration

#### Omit
- Any justification or reasoning (belongs in How / Why)
- Description of what STI / MTG / LTG contain
- Fund background, history, or context-setting

#### Rubric Checklist
- [ ] Allocation percentages stated clearly and sum to 100%
- [ ] Macro regime identified (which scenario is assumed)
- [ ] Reader knows the recommendation within 5 seconds of reading

---

### HOW — Alignment with Fund Objectives (1.5% rubric weight · 15.8% of brief marks)

- **Word count:** 70–90 words (3 bullets)
- **Tables:** 0 — reference unit trust names only, not tabulated
- **Figures:** 0
- **Depth:** MODERATE — connect allocation to each of the three fund objectives; 3 bullets max

#### Must Include
1. **Return target bullet** — state the blended forecast return, compare it to the CPI + 2.5% hurdle (5.2%), and quantify the margin in basis points. Mention fee drag absorption if space permits.
2. **Liquidity bullet** — show STI % satisfies the 10%-within-12-months requirement; STI + MTG satisfies the 25%-within-3-years threshold. Name redemption frequency (daily for STI).
3. **Risk appetite bullet** — state total equity exposure %, note it is consistent with moderate-high tolerance. Reference STI's low-volatility anchor and LTG's long horizon absorbing drawdowns.

#### Citations Required
- **≥1** on the return/risk figures — cite the data source for the blended return or hurdle comparison (e.g., "(RBA, 2026)" for the CPI benchmark, or the capital market assumptions provider)

#### Where the Citation Must Appear
- In the **return target bullet**, attached to the hurdle rate or the forecast return figure

#### Omit
- Detailed unit trust return forecasts (belongs in Why)
- Academic portfolio theory references (Markowitz, Sharpe, etc.) — **hard prohibition**
- Discussion of alternative allocations or scenarios

#### Rubric Checklist
- [ ] Return target addressed with a specific number and compared to hurdle
- [ ] Risk tolerance addressed (qualitative or quantitative)
- [ ] Liquidity requirement addressed with specific thresholds met
- [ ] Link between pool composition (unit trusts) and fund objectives is clear
- [ ] At least 1 credible web-based in-text citation

### WHY — Macro Justification and Asset Class Analysis (7% rubric weight · 73.7% of brief marks)

- **Word count:** 250–290 words (8–10 bullets across 4 sub-headings)
- **Tables:** 0 standalone — **reference Page 2 tables by name** (e.g., "per Table 1" or "see Asset Class Review overleaf")
- **Figures:** 0
- **Depth:** DEEP — this is where 74% of the brief's marks sit; every bullet must carry a specific claim + evidence

#### Sub-section Structure and Must-Include Content

**A. Macro regime (2 bullets, ~70–80 words)**

1. **Australian macro** — CPI figure, RBA policy direction, rate easing timeline. Cite (RBA, 2026).
2. **Global macro** — US inflation rate, Fed stance, IMF global growth projection. Cite (IMF, 2026) or (Federal Reserve, 2026). State the downstream implication: rate cuts support bonds; equities benefit from earnings multiple expansion; AUD depreciation favours unhedged global exposures.

**B. Asset class outlook (2–3 bullets, ~90–100 words)**

3. **Building-block forecast chain** — walk through forecasts respecting the risk-return ordering: cash (3.8%) < bonds (3.9–4.8%) < property (8.4%) < equities (11.7–12.2%) < private equity (13.4%). Cite (Bloomberg, 2026) or the CMA source. **Logic check enforced:** higher risk → higher return within the same regime; if any anomaly exists, explain the mechanism.
4. **Asset-specific drivers** — explain WHY key asset classes perform as forecast:
   - Australian FI: yields compress under easing → 4.8%
   - Global equity: US corporate pricing power + technology earnings resilience + P/E expansion from rate cuts → 12.2% unhedged. **Logic check:** global equity resists inflation drag due to US economic resilience — must state this explicitly
   - Private equity: highest return (13.4%) but highest volatility (20.3%)
   - Infrastructure: defensive, inflation-linked cash flows → 8.6%

**C. Unit trust logic (2 bullets, ~40–50 words)**

5. **Trust composition and forecast** — STI (50% cash / 50% short bond → 4.1% at near-zero vol, pure liquidity vehicle); MTG (45% equity / 35% FI → 7.9%); LTG (60% listed equities, 15% PE, 10% infrastructure → 10.6%). Each trust overweights asset classes forecast to outperform under the assumed regime.

**D. Allocation rationale (1 bullet, ~30–40 words)**

6. **Close the loop** — show the arithmetic: 0.15 × 4.1 + 0.35 × 7.9 + 0.50 × 10.6 = **8.7%** gross return at ~8.1% risk, clearing the 5.2% hurdle for the Fund's moderate-high mandate. LTG maximises real returns over 10 years; MTG balances growth and stability over 5 years; STI buffers near-term obligations.

#### Citations Required
- **≥3** across the Why section, distributed as:
  - ≥1 on macro outlook (RBA Statement on Monetary Policy, IMF WEO, Federal Reserve)
  - ≥1 on asset class forecast data (Bloomberg, Vanguard Capital Markets Model, BlackRock CMA, Refinitiv)
  - ≥1 on a specific market view or global outlook (IMF, S&P Global, JP Morgan Guide to the Markets)
- **All must be credible web-based sources** — NO academic journals, NO textbooks
- **Format:** Org (Year) or (Org, Year) inline

#### Where Citations Must Appear
| Sub-section | Minimum citations | Attach to |
|-------------|:-----------------:|-----------|
| A. Macro regime | 2 | CPI / rate policy claim (RBA); global growth claim (IMF or Fed) |
| B. Asset class outlook | 1 | Forecast return chain or specific asset figure (Bloomberg / CMA provider) |
| C. Unit trust logic | 0 | Numbers derived from trust compositions; no external cite needed |
| D. Allocation rationale | 0 | Arithmetic; no external cite needed |

#### Omit
- Academic portfolio theory (Markowitz, Sharpe, Fama-French, Black-Litterman, risk parity) — **hard prohibition**
- Correlation matrices or diversification theory language
- Historical backtesting results
- Disclaimers, limitations, or caveats (no space; this is a recommendation brief)
- Scenario comparisons — pick one regime and argue for it; do not hedge across scenarios
- Passive voice or hedging language ("may", "could potentially", "it is hypothesised that")

#### Rubric Checklist
- [ ] Macro scenario clearly identified and its differential impact on asset classes explained
- [ ] Logic check passed: high inflation → hurts all assets, but global equity less so (US resilience, pricing power)
- [ ] Logic check passed: higher risk → higher return consistency maintained within the regime
- [ ] Asset class forecasts cited with specific numbers (return %, risk %)
- [ ] Unit trust compositions linked to asset class views under the assumed regime
- [ ] Fund-level allocation connects: macro → asset classes → trusts → STI/MTG/LTG weights → hurdle cleared
- [ ] ≥3 credible web-based in-text citations (no academic sources)
- [ ] CFO-level tone — no jargon, no hedging, no passive voice
- [ ] Every bullet carries a specific claim + evidence; zero filler

## Instructions for the Writing Agent

1. **Draft order: Why → How → What.** Write Why first (analytical core, 74% of marks). Then How (connect to fund objectives). Write What last (summary declaration). This ensures the recommendation in What is consistent with the analysis in Why.

2. **Every bullet = claim + evidence.** No bullet should be pure assertion. Each point must include a specific number, a cited source, or a stated causal mechanism.

3. **Cite as you go.** Format: (Org, Year) inline. No reference list. Minimum 4 citations total: 0 in What, ≥1 in How, ≥3 in Why.

4. **Data to extract and use:**
   - Macro assumptions: CPI 2.7%, RBA easing, US CPI 2.3%, IMF global growth 3.3%
   - Asset class forecasts: Cash 3.8%, Aus FI 4.8%, Global FI 3.9%, Aus Equity 11.7%, Global Equity 12.2%, Property 8.4%, Infrastructure 8.6%, PE 13.4%
   - Trust returns: STI 4.1%, MTG 7.9%, LTG 10.6%
   - Trust compositions: STI (50% cash / 50% short bond), MTG (45% equity / 35% FI), LTG (60% listed equities / 15% PE / 10% infrastructure)
   - Portfolio: 15% STI / 35% MTG / 50% LTG → 8.7% return, ~8.1% risk
   - Hurdle: CPI + 2.5% = 5.2% → exceeded by 350 bps
   - Liquidity: STI 15% covers 10%/12-mo; STI+MTG 50% covers 25%/3-yr

5. **Enforce logic checks:**
   - Risk-return ordering: 3.8% < 3.9% < 4.8% < 8.4% < 8.6% < 11.7% < 12.2% < 13.4% — must hold
   - If showing inflation impact: all assets hurt, but global equity less so (US corporate pricing power, tech earnings resilience)
   - Higher risk → higher return within the same macro regime — flag any anomaly

6. **If the draft exceeds ~400 words:** Cut from How first (lowest marks-per-word). Then trim descriptive bullets in Why sub-section B. Never cut sub-sections A or D of Why.

7. **Tone enforcement:** Replace "it is hypothesised that…" → "data shows". Replace "diversification benefit" → "spreading risk across asset types". Replace "empirical evidence suggests" → "[Source] forecasts". Write for a CFO with 60 seconds.
